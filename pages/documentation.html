Getting Started

Get up and running with Visionade in under 10 minutes. Follow these simple steps to create your first custom report.

Install from Atlassian Marketplace

Installing Visionade is quick and easy through the Atlassian Marketplace:

  1. Go to your Jira instance administration
  2. Navigate to "Find new apps" in the left sidebar
  3. Search for "Visionade" in the marketplace
  4. Click "Install" and accept the permissions
  5. Wait for the installation to complete (usually under 30 seconds)
Tip: You'll need Jira administrator permissions to install apps. If you don't have access, contact your Jira admin.

Setting Up Visionade

Once installed, configure Visionade for your first report:

  1. Click on "Apps" in the navigation bar
  2. Select "Visionade" in Apps section

Configure Dimensions

Configure your dimensions to use in reports:

  1. Open Data Sources tab, select a data source to open
  2. Select dimensions by checking the checkbox of each, then click ‚ÄúSave Selected‚Äù
  3. Index all dimensions to have last updated data
Tip: You can create custom dimensions from fields in Data Source, and grouping their values based on data type.

Configure Data Set

Set up your data configuration for reporting:

  1. Select your primary data source
  2. Select Dimensions based on the selected data source
  3. Type JQL query to narrow down data from data source
  4. Preview data based on data set configuration
Important: If you leave JQL query blank, it will query data with the following query: project is not empty

Configure Report

Create custom reports with drag-and-drop simplicity:

  1. Drag fields to appropriate section
  2. Select aggregations for Values dimensions (sum, average, count, min/max)
  3. Filter by dimensions
  4. Choose your chart type (bar, column, pie, line)
  5. Click "Apply Config"
  6. Preview and adjust your report
Pro Tip: Start with simple bar charts before moving to more complex visualizations.

Add Reports to Jira Dashboard

Integrate your reports directly into Jira dashboards:

  1. Go to your Jira dashboard
  2. Click "Add gadget"
  3. Select "Visionade Report" from the list
  4. Choose your configured report
  5. Adjust size and position as needed

Troubleshooting & Support

Always getting supported

Common Issues

Report Not Loading
Issue: Report shows "Loading..." or displays no data

Solutions:

  1. Verify data source permissions
  2. Check if filters are too restrictive
  3. Ensure data exists for the selected time period
  4. Try refreshing the page
Chart Display Issues
Issue: Charts appear blank or incorrectly formatted

Solutions:

  1. Check field mappings in report configuration
  2. Verify calculation settings
  3. Test with default chart settings
  4. Clear browser cache if needed
