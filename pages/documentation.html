Quick Start Guide

Get up and running with Visionade in under 10 minutes. Follow these simple steps to create your first dashboard.

Install from Atlassian Marketplace

Installing Visionade is quick and easy through the Atlassian Marketplace:

  1. Go to your Jira instance administration
  2. Navigate to "Find new apps" in the left sidebar
  3. Search for "Visionade" in the marketplace
  4. Click "Install" and accept the permissions
  5. Wait for the installation to complete (usually under 30 seconds)
Tip: You'll need Jira administrator permissions to install apps. If you don't have access, contact your Jira admin.

Create Your First Dashboard

Once installed, you'll see Visionade in your Jira apps menu:

  1. Click on "Apps" in the top navigation
  2. Select "Visionade" from the dropdown
  3. Click "Create New Dashboard"
  4. Give your dashboard a name (e.g., "Sprint Overview")
  5. Choose your data source (Jira project or Table Grid)
Dashboard Name: "My First Dashboard" Data Source: "PROJECT-KEY" or Table Grid instance Template: Sprint Performance (recommended for beginners)

Connect Your Data Sources

Visionade can pull data from multiple sources:

  • Jira Projects: Select one or more projects to aggregate data
  • Table Grid: Connect to existing Table Grid instances
  • Custom JQL: Use advanced Jira Query Language for specific filters
Important: Make sure you have view permissions for all projects you want to include in your dashboard.

Add Your First Chart

Now for the fun part - creating visualizations:

  1. Click "Add Chart" in your dashboard
  2. Choose a chart type (Bar Chart is great for beginners)
  3. Drag "Issue Status" to the X-axis
  4. Drag "Issue Count" to the Y-axis
  5. Click "Apply" to generate your chart
Pro Tip: Use the "Auto-suggest" feature to get recommended chart configurations based on your data.

Share Your Dashboard

Make your insights accessible to your team:

  1. Click the "Share" button in the top-right corner
  2. Choose sharing permissions (View or Edit)
  3. Add team members by email or Jira username
  4. Set up automatic email reports (optional)
  5. Generate a public link for external sharing

Chart Types & Configurations

Visionade supports multiple chart types to visualize your data effectively:

Bar Charts

Perfect for comparing categories or showing distributions:

  • Horizontal and vertical orientations
  • Stacked and grouped options
  • Custom color schemes

Line Charts

Ideal for showing trends over time:

  • Single or multiple data series
  • Smooth or stepped line styles

Pie Charts

Great for showing parts of a whole:

  • Donut chart variations
  • Exploded slices for emphasis
  • Percentage or value labels

Create advanced metrics without writing code:

Export Options

  • Excel: Raw data for further analysis
  • PNG/JPG: Images for documentation
  • CSV: Data export for other tools

Table Grid Integration

Visionade's unique integration with Table Grid provides enhanced capabilities:

Setup Process

  1. Ensure Table Grid is installed in your Jira instance
  2. Navigate to Visionade settings
  3. Click "Connect Table Grid"
  4. Select the Table Grid instances to connect
  5. Configure data synchronization settings

Benefits

  • Enhanced data visualization options
  • Cross-reference Jira and Table Grid data
  • Advanced filtering and sorting
  • Real-time data synchronization

API Integration

Connect Visionade with external systems (Enterprise plan):

  • REST API for data export

  • Custom data connectors

Troubleshooting

Common Issues & Solutions

Dashboard Not Loading

Issue: Dashboard shows "Loading..." indefinitely

Solutions:

  1. Check your internet connection
  2. Verify you have permissions to access the data sources
  3. Try refreshing the page (Ctrl+F5)
  4. Clear browser cache and cookies
  5. Contact support if the issue persists

Charts Not Displaying Correctly

Issue: Charts appear blank or show incorrect data

Solutions:

  1. Verify your data source has data for the selected time period
  2. Check field mappings in chart configuration
  3. Ensure filters aren't too restrictive
  4. Try recreating the chart with default settings

Performance Issues

Issue: Slow loading times or timeouts

Solutions:

  1. Reduce the date range for your queries
  2. Limit the number of data points in charts
  3. Use filters to reduce dataset size
  4. Consider upgrading your Jira instance

Best Practices

  • Use Filters: Apply filters to focus on relevant data
  • Regular Updates: Schedule regular data refreshes
  • Team Training: Provide training for all dashboard users
  • Performance: Monitor dashboard performance and optimize as needed

API Documentation

Available with Enterprise plan

Authentication

All API requests require authentication using API tokens:

curl -H "Authorization: Bearer YOUR_API_TOKEN" \ -H "Content-Type: application/json" \ https://your-domain.atlassian.net/rest/visionade/1.0/dashboards

Endpoints

  • GET /dashboards - List all dashboards
  • POST /dashboards - Create new dashboard
  • GET /dashboards/{id}/data - Export dashboard data
  • POST /webhooks - Configure webhooks
