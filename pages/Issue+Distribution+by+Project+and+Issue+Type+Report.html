ðŸš€ Step-by-Step Implementation

Step 1: Configure Jira Data Source

1

Open Data Sources tab, click on Jira source

2

Configure Jira connection settings if not already connected

3

Click Select Dimensions button

4

Select key dimensions: Project, Issue Type, Status, Priority, Assignee, and Created Date

5

Click Save Selected to save the dimension configuration

6

Click Index All Dimensions button to reindex the data of all selected dimensions, ensuring it is fully up to date.

Step 2: Configure Data Set

1

Open Data Sets tab, click Create Data Set button

2

Enter "Issue Distribution Data" for Data Set Name

3

Select "Jira" in Select Data Sources section

4

In Select Project field, select "All Projects" or specific projects you want to analyze

5

Enter JQL query to filter the issues you want to analyze:

project in (PROJ1, PROJ2, PROJ3) AND created >= -30d

Then click "Run Query" or "Preview" button to filter and preview data in Data Preview section

6

Click Save to save data set configuration

Step 3: Configure Issue Distribution Report

1

Open Reports tab, click Create Report button

2

Enter "Issue Distribution by Project and Type" for Report Name and click Next button

3

In Table View tab, select data set "Issue Distribution Data" that we have configured

4

Open Analytics & Pivot tab to start configuring report

5

Drag dimension Project to section Rows to view project names as row headers

6

Drag dimension Issue Type to section Columns to view issue types as column headers

7

Drag dimension Issue Key to section Values to count issues in each cell

Select Count in Aggregation field

8

Drag dimension Status to dimension Filters to filter by specific statuses

Click on Status dimension to select filter conditions like "Status Is In" and choose relevant statuses

9

Drag dimension Created Date to dimension Filters to filter by date range

Click on Created Date dimension to select filter condition Date Is After and enter value "2024-01-01"

ðŸ“… Date Format: Use format YYYY-MM-DD

10

Select Bar Chart or Heatmap in section Visualization for better visual representation

11

Click Apply Config to generate report

12

Click "Save" to save report configuration

Step 4: Add Issue Distribution Report to Jira Dashboard

1

Open Dashboards screen and click Create dashboard button

2

Enter "Project Management Dashboard" for Name and click Save

3

Click Edit button to edit the dashboard

Search "Visionade" to search gadget

Click Add button on Visionade Report gadget

4

Select Issue Distribution by Project and Type Report

5

Click "Save Changes" to save changes on gadget

6

Click "Change layout" and select "Two columns" for better visualization

7

Click Done to save dashboard
