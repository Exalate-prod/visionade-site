ðŸš€ Step-by-Step Implementation

Step 1: Configure Table Grid Data Source

1

Open Data Sources tab, click on Table Grid source

2

Click Select on grid "Order Burger Demo"

3

Click Select Dimensions button

4

Select dimensions and then click Save Selected

5

Click Index All Dimensions button to reindex the data of all selected dimensions, ensuring it is fully up to date.

Step 2: Configure Data Set

1

Open Data Sets tab, click Create Data Set button

2

Enter "Burger Orders" for Data Set Name

3

Select "Table Grid" in Select Data Sources section

4

In Select Grid field, select grid "Order Burger Demo"

5

Enter JQL query to narrow down the data set that you want:

key = (HNGTST-33, HNGTST-89, DU2-254, B2-43)

Then click "Run Query" or "Preview" button to filter and preview data in Data Preview section

6

Click Save to save data set configuration

Step 3: Configure Revenue Report

1

Open Reports tab, click Create Report button

2

Fill in Report Name and click Next button

3

In Table View tab, select data set "Burger Orders" that we have configured

4

Open Analytics & Pivot tab to start configuring report

5

Drag dimension Name to section Rows to view menu item name as row headers

6

Continue to drag dimension Size to section Rows and below dimension Name to view size of each menu item

7

Drag dimension Order Type to dimension Columns to view order types as column headers

8

Drag dimension Total to section Values to view revenue in each field of the table

Select Sum in Aggregation field

9

Drag dimension Order at to dimension Filters to filter orders in 2025

Click on Order at dimension to select filter condition Date Is After and enter value "2025-01-01"

ðŸ“… Date Format: Use format YYYY-MM-DD

10

Select Bar Chart in section Visualization

11

Click Apply Config to generate report

12

Click "Save" to save report configuration

Step 4: Add Revenue Report to Jira Dashboard

1

Open Dashboards screen and click Create dashboard button

2

Fill in Name and click Save

3

Click Edit button to edit the dashboard

Search "Visionade" to search gadget

Click Add button on Visionade Report gadget

4

Select Revenue by Menu Items Report

5

Click "Save Changes" to save changes on gadget

6

Click "Change layout" and select "One column"

7

Click Done to save dashboard
